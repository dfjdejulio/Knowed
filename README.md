Knowed
===

This is intended to be a Framework that helps turn JavaScriptCore into a
useful non-web runtime.  It should remind folks of "node.js".


Notes
---

Here are some built-in objects already in JavaScriptCore.

* Math
* JSON
* Number
* RegExp
* Boolean
* String
